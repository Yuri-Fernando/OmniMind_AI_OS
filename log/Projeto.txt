🧠 Projeto Inicial
Assistente Inteligente Multimodal com RAG, Guardrails e Voz
Um agente de IA capaz de:
responder perguntas sobre documentos
conversar por voz
usar ferramentas
evitar prompt injection
avaliar qualidade das respostas

🧠 Funcionalidades
1️⃣ Agente com ferramentas

Exemplo:
Usuário pergunta:
"Qual o clima em Campinas?"
O agente usa API de clima.

2️⃣ Pipeline RAG
Usuário pergunta:
"Explique o relatório técnico."
Fluxo:
documento → chunking → embeddings → busca → LLM

3️⃣ Guardrails

Proteção contra:
prompt injection
jailbreak
conteúdo sensível

4️⃣ Avaliação automática

Usar:
Ragas
DeepEval
Para medir:
relevância
precisão
groundedness

5️⃣ Observabilidade

Logs com:
Langfuse
OpenTelemetry

🎯 Resultado final
✔ conversa por texto
✔ conversa por voz
✔ responde usando documentos
✔ usa ferramentas
✔ evita ataques de prompt injection
✔ mede qualidade das respostas

Essa arquitetura simula o que empresas fazem:

Planner agent → decide o que fazer
RAG agent → busca conhecimento
Tool agent → executa APIs
Memory manager → guarda histórico
LLM router → escolhe modelo
